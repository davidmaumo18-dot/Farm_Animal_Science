Technical
How This Was Built

A complete data pipeline from raw Excel data to structured SQL database to Python analysis.

Excel
Data collection template with 5 sheets, dropdown validation, auto-calculating formulas
Python
pandas for data loading & cleaning, mysql-connector for database insertion, matplotlib for charts
MySQL
5-table relational database with JOIN queries for cross-table health and production analysis
Jupyter
Full analysis notebook combining SQL queries, data visualization, and written findings
