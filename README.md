# Farm Animal Science Dashboard

A full-stack data analytycs project tracking a 32-animal mixed farm herd.

## What This Project Does
-Tracks weight and flags underweight/overweight animals by species
-Ranks dairy Cows by average daily milk_yield
-Monitors vaccination schedules and health events
-Analyses feed costs and spending trends over time

## Tools Used
-*Python* (pandas, matplotlib, mysql-connector)
-*MySQL 8* _ 5-table relational database
-*Jupyter Notebook* _ full analysis pipeline
-*Excel* _ data collection template with validation

## Files
-`Farm_Animal_Science_Portfolio.html` _interactive portfolio page
-`Animal_Science.ipynb` _ Jupyter analysis notebook
-`Farm_Sample_Data_Filled-1.xlsx` _ sample dataset(32 animals)

## Key Findings
1.97% of animals are within healthy weight ranges
2.Clara (Ayshire cow) is the top milk producer at 16.1L/day
3.Feed spending peaks 18% above average in March_May dry season
